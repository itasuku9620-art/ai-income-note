## プロンプト集・テンプレート

### 1. ターゲット設定最適化プロンプト
```
あなたはAI副業コンサルタントです。以下の条件でターゲットを詳細に設定してください：

【業界・ジャンル】：[あなたの専門分野]
【想定年齢層】：[20-30代など]
【課題・悩み】：[具体的な痛み]

以下の形式で出力：
- ペルソナ設定（名前、年齢、職業、年収、悩み）
- 購買動機（なぜお金を払うのか）
- 競合分析（既存解決手段の問題点）
- 価格感度（いくらまでなら払うか）
```

### 2. 売れるタイトル生成プロンプト
```
以下の条件で、noteで売れやすい記事タイトルを10個作成してください：

【メインキーワード】：[AI副業など]
【ターゲット】：[具体的な読者像]
【提供価値】：[得られる結果・ベネフィット]

条件：
- 数字を含む（期間、金額、ステップ数など）
- 具体的な結果を明示
- 緊急性・限定性を演出
- 40文字以内
- 検索されやすいキーワードを含む
```

### 3. 高品質記事構成作成プロンプト
```
以下のテーマで、noteで売れる記事構成を作成してください：

【記事テーマ】：[具体的なテーマ]
【ターゲット読者】：[詳細なペルソナ]
【文字数】：2000-3000文字
【価格設定】：[300-980円]

構成要件：
- 無料部分で価値提供（読者の課題共感+解決策の概要）
- 有料部分で実用性（具体的手順、テンプレート、プロンプト）
- SEOを意識した見出し構成
- 読了率を高める仕掛け
```

【背景設定】：[年齢、職業、経験レベル]

以下の要素を含めて作成：
- Before（開始前の状況・課題）
